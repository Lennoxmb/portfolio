Konami Code

In this project, we created a webpage that displays a surprise when the user enters the Konami Code. The Konami Code is a cheat code that appears in many Konami video games. The code is up, up, down, down, left, right, left, right, B, A, and start, in this case, Enter. We learned about Event Listeners in JavaScript and were to make an alert come up stating the user "You have added 30 lives!" along with any customizations we wanted.
