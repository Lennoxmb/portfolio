Coffee Project ☕

This was a pared program, and I worked with Marc Mata. In this project, it was necessary to change an HTML page so that information on coffee would be presented using divs rather than tables. According to their IDs, the coffees are listed on the page in ascending order. Additionally, it is now possible to search for specific coffees by name and display results that match. Users can also add new coffees by completing a form.
